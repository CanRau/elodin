# Fela

## Configuration

| Option        | Type    | Default | Description                                                                               |
| ------------- | ------- | ------- | ----------------------------------------------------------------------------------------- |
| reactFela     | Boolean | `false` | Generate React components using react-fela's `createComponent` API instead of plain rules |
| dynamicImport | Boolean | `false` | Enables dynamic inline `import()` calls to lazy-load static CSS files                     |
